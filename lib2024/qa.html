LIB2024- QA

  1. Qus: What are the three componets of Web?

    Ans: HTML, CSS, Javascript.

  2. Qus: Define version control and give one example.

    Ans: A system that records changes to a set of files over time so that specific versions can be referred to. An example of this is GitHub.

  3. Qus: What tag is used to create the title of a web page? Give an example.

    Ans: The <title> tag is used to create the title of a web page.

  4. Qus: What tag is used to create a paragraph in HTML? Give an example.

    Ans: The <p> tag is used to create a paragraph in HTML.

  5. Qus: Identify the odd one from these tags: <span>, <p>, <a>, <i> and state the reason?

    Ans: The <p> tag is the odd one out, because it formats text instead of transforming it.

  6. Qus: What tags are required to create a navigation list in HTML?

    Ans: <ul> and <li> are used to create a navigation list in HTML.

  7. Qus: What element is used to create images? Name its attributes.

    Ans: <img> is used to create images, and its attributes are src and alt.

  8. Qus: Identify the odd one from these tags: <table>, <tr>, <b>, <td> and state the reason.

    Ans: <b> is the odd tag out, because it bolds text instead of creating a table like the other ones do.

  9. Qus: Give an example of color picker input.

    Ans: <input type="color">

  10. Qus: What is the use of name and value attributes? Give two examples.

    Ans: The name attribute specifies a name for an element, while the value specifies the value of an element.
    Ex. <input type="text" name="example" value="human">

  11. Qus: Give an example of inline style.

    Ans: <p style="color:red;font-size:20px;">This is an example</p>

  12. Qus: How would you link an external style sheet? Give an example.

    Ans: <link rel="stylesheet" type="text/css" href="mystyle.css">

  13. Qus: Create a CSS block to center an element both horizontally and vertically that has position:absolute and has a width:200px and height:200px.

    Ans: #containter {
    position: relative;
    }
    .center {
    margin: auto;
    position: absolute;
    width: 200px;
    height: 200;
    }

  14. Qus: Create a CSS keyframes block called "spin" that rotates an element.

    Ans: .loader {
    animation: spin 0.25s linear infinite;
    }

  15. Qus: Create a CSS animation block to rotate an element that has the following properties:

    • class "rotate-me"
    • It rotates at intervals of 0.5s with "linear" transition using animation keyframes "spin"

    Ans: .rotate-me {
    animation: spin 0.5s linear 5;
    }
    @keyframes spin {
    from { transform: rotate(0deg); }
    to { transform: rotate(360deg); }
    }

  16. Qus: How would you create an alert with the string "I love JavaScript" using Javascript>

    Ans: function myFunction() {
    alert("I love JavaScript");
    }
